# Mandala Tattoo #

## What is a Mandala? ##
:diamond_shape_with_a_dot_inside: A **mandala** is a geometric diagram/art piece originating in Hindu and Buddhist cultures.

## How I Made It ##
:diamond_shape_with_a_dot_inside: For my output, I utilized various geometric shapes to form different kinds of designs. My design is based on a tattoo, so no color is added.

## How I Came Up With This Design ##
:diamond_shape_with_a_dot_inside: This simple mandala is inspired by a dreamcatcher tattoo that my cousin has.
